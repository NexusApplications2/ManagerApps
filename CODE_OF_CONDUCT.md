# Código De Conduta

Este projeto espera colaboração respeitosa e objetiva.

## Comportamentos Esperados

- Seja claro e direto.
- Critique código e decisões, não pessoas.
- Explique contexto técnico quando discordar.
- Ajude novos contribuidores a entender o fluxo do projeto.
- Não publique dados sensíveis de terceiros.

## Comportamentos Inaceitáveis

- Assédio, ataque pessoal ou discriminação.
- Divulgação de tokens, senhas ou dados privados.
- Spam, autopromoção fora de contexto ou conteúdo malicioso.
- Pressionar mantenedores por prazo sem contribuir com contexto ou solução.

## Aplicação

Mantenedores podem editar, ocultar ou remover comentários, issues e pull requests que violem este código.
